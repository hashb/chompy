Learning how to browse the web faster and more easily

Learning a few basic things, such as how to use the buttons on the Internet Explorer toolbar, can make browsing the web easier and faster.

To start browsing the web, click any link on your home page, which is the page that appears when you open Internet Explorer. You can see whether an item on a page is a link by moving the mouse pointer over the item. If the pointer changes to a hand, the item is a link. A link can be a picture, a 3-D image, or colored text (usually underlined). If you press the CTRL key when you click, the link will open in a new tab. Click the new tab to view the webpage.

Now you're ready to find a website on your own.

To open a webpage or folder, or to run a program

In addition to surfing the web, Internet Explorer lets you run programs and access files and folders on your computer.

  • To go to a webpage, type the Internet address (for example, www.microsoft.com) in the Address bar, and then press ENTER.
  • To run a program from the Address bar, type the program name, and then press ENTER.
  • To browse through folders from the Address bar, type the drive letter or drive letter and folder name (for example, C:\ or C:\My_Novel), and then press ENTER. The folder contents will be displayed in a new window.

To find a webpage about a specific topic

You can find a great deal of information on a wide range of topics by searching the Internet.

To search for a specific topic

  1. Click in the search box in the upper right corner of Internet Explorer.
  2. Type a keyword (for example, "dogs"), and then press ENTER.

You can choose the search provider that you want Internet Explorer to use.

To search by using a different search provider

  1. Click the Search Options button (the arrow to the right of the search box), and then select a different provider.

    2. Your search will be repeated with the new provider.

To change the default search provider

  1. Click the Search Options button (the arrow to the right of the search box), and then click Change Search Defaults.
  2. Select the provider you want to use for searches, click Set Default, and then click OK.

To return to webpages you recently visited

  • To return to the last page you viewed, click the Back button.
  • To return to the page you were on before you clicked the Back button, click the Forward button.
  • To see a list of the last few pages you visited, click the Recent Pages button . The Recent Pages button is the small arrow to the right of the Forward button.

To go to a specific webpage

  • To return to the page that appears each time you open Internet Explorer, click the Home button .
  • To select a webpage from your list of favorites, click the Favorites Center button .
  • To select a webpage from the list of those you visited recently, click the Recent Pages button . The Recent Pages button is the small arrow to the right of the Forward button.

What to do if a webpage isn't working

  • If a page you are trying to view is taking too long to open, click the Stop button on the right side of the Address bar. If the page has stopped loading, you can click the Refresh button to try again.
  • If you get a message that a webpage cannot be displayed, or you want to make sure you have the latest version of the page, click the Refresh button .
